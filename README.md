# CryptoVault – Web3 Wallet

CryptoVault is a simple Web3 wallet interface built with React + Vite.

## Features
- Create Ethereum wallet
- Import wallet
- Send ETH
- Receive ETH
- Transaction history
- Sepolia testnet support

## Live Demo
https://crypto-vault-neon.vercel.app/

## Tech Stack
- React
- Vite
- JavaScript
- Ethers.js
- Sepolia Testnet
